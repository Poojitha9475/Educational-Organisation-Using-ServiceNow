# Admission Form Design

## Sections
- **Admission Details**
  - Student Name
  - Grade
  - Admission Number
  - Admission Status

- **Address Details**
  - Address
  - City
  - State
  - Pincode

- **Parent Details**
  - Father Cell
  - Mother Cell

## Layout
- Two-column layout for clean alignment and readability

## Notes
- Admission Number is system-generated  
- Admission Status is workflow-controlled
